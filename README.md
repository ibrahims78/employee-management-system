# Staff Health Analyzer | محلل صحة الموظفين 🏥📊

### [English Description]
A specialized tool designed to analyze and monitor the health status of staff members. This project aims to provide data-driven insights to improve workplace wellness and maintain health records efficiently.

**Key Features:**
* 📈 Data visualization for health metrics.
* 📂 Staff health record management.
* 🔍 Advanced filtering and analysis.
* 🚀 Built & Deployed via Replit.

---

### [الوصف باللغة العربية]
أداة متخصصة مصممة لتحليل ومراقبة الحالة الصحية للموظفين. يهدف هذا المشروع إلى تقديم رؤى مستندة إلى البيانات لتحسين الصحة العامة في بيئة العمل وإدارة السجلات الصحية بكفاءة.

**الميزات الرئيسية:**
* 📈 تمثيل بياني للمؤشرات الصحية.
* 📂 إدارة سجلات الموظفين الصحية.
* 🔍 أدوات متقدمة للفلترة والتحليل.
* 🚀 تم التطوير والنشر بواسطة Replit.
